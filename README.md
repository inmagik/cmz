# cmz
Our kick-ass django based cms.

## Install 

Grab this repo or install from github.

```
pip install git+ssh://git@github.com/inmagik/cmz.git#egg=cmz
```

Recommended way: use virtualenv to install and run a website

## Develop locally

Start a new website:

```
cmz_start.py
```

And run it locally:

```
cmz_play.py [port]
```
